ScrollSpark

A lightweight scroll-trigger animation library using IntersectionObserver.

Install via npm install scrollspark or include from a CDN:

<link rel="stylesheet" href="https://unpkg.com/scrollspark/dist/scrollspark.css">
<script src="https://unpkg.com/scrollspark/dist/scrollspark.umd.js"></script>

Getting started

Create an instance and register elements to animate:

import ScrollSpark from 'scrollspark';

const ss = ScrollSpark.init();

// animate all elements with class "fade-me" using built-in effect
ss.on('.fade-me', { animate: 'fade-up' });

Attribute API

You can also use data attributes directly in your markup:

<div ss-animate="fade-up" ss-delay="200">Hello</div>

Other helpers

  • ss.addClass(selector, className, options)
  • ss.stagger(selector, options) – stagger siblings
  • ss.progress(selector, callback) – receive scroll ratio
  • ss.trackNav(navSelector, activeClass) – highlight links
